#  Bataille Navale - Projet Final

# Objectif
Un mini-jeu de **bataille navale** codé en Python, avec tests unitaires et gestion Git.

# Prérequis
- Python 3.8 ou plus
- `pip` installé

# Installation
```bash
git clone <ton lien GitHub>
cd bataille-navale
python -m venv .venv
source .venv/Scripts/activate  # sous Windows Git Bash
pip install -r requirements.txt
